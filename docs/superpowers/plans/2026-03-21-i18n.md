# Internationalization Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add multi-locale support (en-us, en-gb, de-de) with fully independent content per market, a language picker, and first-visit locale detection redirect.

**Architecture:** Astro's built-in i18n routing with `prefixDefaultLocale: false`. UI strings in TypeScript files per locale. Content collections with locale subdirectories. A shared `Layout.astro` component holds all `<head>` boilerplate, theme logic, hreflang tags, and the locale redirect script. Components receive string sections as props.

**Tech Stack:** Astro 6 i18n, TypeScript, SCSS

**Spec:** `docs/superpowers/specs/2026-03-21-i18n-design.md`

import type { Strings } from './types';
import { enUs } from './en-us';
import { enGb } from './en-gb';
import { deDe } from './de-de';

export type Locale = 'en-us' | 'en-gb' | 'de-de';

export const locales: Locale[] = ['en-us', 'en-gb', 'de-de'];
export const defaultLocale: Locale = 'en-us';

const strings: Record<Locale, Strings> = {
  'en-us': enUs,
  'en-gb': enGb,
  'de-de': deDe,

export function getStrings(locale: string | undefined): Strings {

### Task 5: Create Layout.astro

**Files:**
- Create: `src/layouts/Layout.astro`

- [ ] **Step 1: Create src/layouts/Layout.astro**

Extract all shared `<head>` content from `src/pages/index.astro` into a layout component. The layout handles:

- `<html lang={htmlLang}>` — set from `localeConfig[locale].htmlLang`
- `<head>` with meta charset, viewport, Google Fonts links, IBM Plex Mono font-face
- `<title>` and `<meta name="description">` from props
- Theme detection inline script (copy from current index.astro)
- Locale redirect inline script (conditionally, via `includeRedirectScript` prop)
- `hreflang` alternate link tags (from `alternates` prop)
- Global CSS: resets, `html.dark`/`html.light` theme variables, `.reveal` animations, body styles
- `<slot />` for page content

Props interface:

```ts
type Props = {
  locale: Locale;
  title: string;
  description: string;
  alternates?: Array<{ hreflang: string; href: string }>;
  includeRedirectScript?: boolean;
};
```

The redirect script (included only when `includeRedirectScript` is true):

```html
<script is:inline>
(function() {
  if (localStorage.getItem('locale')) return;
  var lang = (navigator.language || '').toLowerCase();
  var target;
  if (lang.startsWith('de')) target = 'de-de';
  else if (lang === 'en-gb') target = 'en-gb';
  if (target) {
    localStorage.setItem('locale', target);
    window.location.replace('/' + target + '/');
  } else {
    localStorage.setItem('locale', 'en-us');
  }
})();
</script>
```
